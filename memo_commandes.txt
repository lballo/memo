git branch => connaitre le nom de la branche sur laquelle on se trouve
git init => création d'un repo
Pour créer une nouvelle branche: git branch nom_de_la_branche
par défaut se positionne sur la branche Master mais une nouvelle branche a été créée
pour se positionner sur la nouvelle branche: git checkout nom_de_la_branche
lorsqu'on fait git branch par la suite l'étoile doit se trouve devant nom_de_la_branche
pour revenir sur master: git checkout master
